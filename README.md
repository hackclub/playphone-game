# Playphone Round 7

This is a Game made for the seventh round of #playphone on the Hack Club Slack! Playphone is a community made game where others will contribute to this game in rounds, and at the end of it all, we will have an extremely polished game.

Changelog for Round 7:

- Changed end scene to bring back to menu instead of killing game
- Got rid of MainMenue.tscn (unused)
- got rid of Mainmenu.tscn (unused)
- Changed levels menu to be dynamic/to use gridcontainer for consistency
- Added a stopwatch along with personal bests and end time to end screen
- 67
- Level 4
- Made use of unused animations and added (and also refactored the system) animations for dashing, idling, jumping/falling
- Camera now zooms out based on vertical speed
- Added E to controls
- Settings page redo, converted from control node to CanvasLayer for proper sizing and no stretch, set volume slider to current volume on ready, added percentage display
- Pressing exit on web now shows a "its now safe to turn off your PC" scene instead of kabooming
- Added button to clear all PBs in settings
- Adjusted character hitbox to be slightly smaller and more consistent so the character doesn't levitate and the cloud bounces count as hitting the ground for dash reset
- Level 4 is now longer, awesomer, stronger, and also added grass
